Student Course Registration System

Project Description

This project is a backend system for a TVET training center.
It allows students to register, couses to be added, and students to be enrolled in courses.

Technologies Used
-Backend:Node.js + Express
-Database: MYSQL
-Database Name: studentregistration
-Testing Tool: Postman

Features
-Student registration
-Course management
-Student enrollment
-View enrolled courses


Database Design
The system  uses three tables:
-students
-courses
-Enrollments

API Endpoints

Students
-POST /Students - Register a student
-GET /Students - List Students

Courses

-POST /Courses - Add a course
-GET /Courses - List Courses

Enrollments

-POST /enrollments -Enroll student
-GET /enrollments - View enrollments

How to Run the Project
1.Clone the repository
2.Install dependencies
3.Create database `studentregistration`
4.Run the server
5.Test APIs using Postman
